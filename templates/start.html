Welcome to Squarewheel

This is an service allows you look up the accessibility of venues from Foursquare. It does so by searching wheelmap.org for the venue. It also allows you to change the accessibility status of a venue.

You can search for places or log in with your account to look up your last checkins and todo list.

This project is Open Sorce and you can fork it on Github
